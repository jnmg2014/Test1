# Three.js Block World Demo

## Running locally

1. Install dependencies with `npm install`.
2. Start the development server using `npm run dev` and open the provided URL.

### Running without Node.js

Thanks to the import map defined in `index.html`, you can also open the project
directly from the filesystem in any modern browser. Simply double-click
`index.html` and the scene will load using CDN-hosted Three.js modules.
